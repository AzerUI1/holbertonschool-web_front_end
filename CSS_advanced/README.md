CSS (Cascading Style Sheets) is used to style and format the appearance of HTML elements. With CSS, you can:

- Change colors, fonts, and text size  
- Add spacing and layouts  
- Create responsive designs for all devices  
- Add animations and transitions  
- Style forms, buttons, and images  

CSS works together with **HTML** (structure) and **JavaScript** (interactivity).
